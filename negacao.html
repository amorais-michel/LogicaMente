Negação


Qualquer proposição pode ser negada.


Quando negamos uma proposição, isso significa que estamos mudando o seu valor lógico, ou seja, invertendo ele. Se uma proposição é verdadeira, ela vira falsa. Se uma proposição é falsa, ela vira verdadeira.

Usamos o simbolo ~ (til) para representar uma negação

Podemos expressar uma negação como (não p) ~p Normalmente, podemos ler a notação acima como:

Não p

Não é verdade que p

É falso que p


Exemplos:


a) p: Hoje é domingo

~p(não p): Hoje não é domingo (proposição p negada)


Note que no exemplo acima, temos uma proposição p que afirma que é domingo, e abaixo dela, temos a sua negação, que afirma que hoje não é domingo. A negação trabalha dessa forma, ela nega e muda o valor lógico das coisas.


b) q: Brasil é um país (V)
~q (não q) : Não é verdade que Brasil é um país (F)


c) r: Moro no Rio Grande do Sul (V)
~r (não r): Não moro no Rio grande do Sul (F)


d) s: O céu é azul (V)
~s : O ceu não é azul (F)

e) t: O Gremio é o maior time do Rs (F)
~t: O Gremio não é o maior time do Rs (V)
